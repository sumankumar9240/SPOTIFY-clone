# 🎵 Spotify Frontend Clone

A modern and responsive Spotify-inspired frontend clone built using HTML and CSS.  
This project recreates the look and feel of the Spotify Web Player with a clean and user-friendly interface.

---

## 🚀 Features

- 🎧 Spotify-inspired UI Design
- 📱 Responsive Layout
- 📂 Sidebar Navigation
- 🎵 Music Player Section
- ❤️ Playlist & Album Cards
- 🎨 Modern CSS Styling
- ⚡ Clean User Experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome
